Waveform — what to look for

PatternMeaningAction
Smooth wave Normal signal —
Flat top/bottom (clipping) Signal exceeds ±1 Lower vol or limiter threshold
Sharp vertical spikes Click / pop (discontinuity) Check envelope or buffer boundaries
Flat line at zero Silence — no source playing Check note/seq is running
Tiny noise, peak < 1e-8 Possible denormals Check flush-to-zero in DSP code

Spectrum — what to look for

PatternMeaningAction
Fundamental + evenly spaced peaks Normal harmonic series —
Steep rolloff after a frequency Filter is working Check cutoff matches expectation
Flat energy across all bands Noise or excessive distortion Reduce drive / saturation
Bump near 0 Hz (left edge) DC offset Filter bug or feedback accumulation
Energy at right edge (Nyquist) Aliasing Check anti-aliasing in oscillator
Non-integer harmonic spacing FM / ring mod / inharmonic Normal for FM synth
